```
GAIN:   Low

```

## 🎛️ HiBy FC5 操作

### 按键功能
| 按键 | 功能 |
|------|------|
| 功能键 (顶部) | 短按：开屏/切换功能页 |
| 音量+ (中间) | 增加音量 / 上下切换选项 |
| 音量- (底部) | 减小音量 / 上下切换选项 |

### 设置切换
1. 短按**功能键**进入设置界面
2. 按**音量+/-**切换选项
3. 依次循环：音量 → GAIN → FLTR → EQ → 音量

## 📊 关键参数速查

### VULKAN 2 规格
- **阻抗**: 35Ω（易驱动）
- **灵敏度**: 113dB/mW（高灵敏度）
- **频响**: 5Hz - 40kHz
- **单元**: 2DD + 6BA（双动圈六动铁）
- **推荐增益**: Low（避免底噪）

### HiBy FC5 规格
- **DAC 芯片**: Dual CS43198
- **最高支持**: PCM 768kHz / DSD512
- **输出电压**: 2.0Vrms (SE) / 4.0Vrms (BAL)
- **输出阻抗**: <1Ω (SE) / <2Ω (BAL)

## ⚠️ 每日提醒

### 开始听音乐前
1. ✅ 先将 FC5 音量调至最低
2. ✅ 连接 USB-C 线缆
3. ✅ 插入耳机（不要先戴上）
4. ✅ 从低音量开始，逐渐调高

### 听音乐时
- 🔊 建议音量不超过 60-70%
- ⏰ 每小时休息 10-15 分钟
- 🎼 优先选择无损音质
- 🔇 不要在嘈杂环境过度提高音量

### 结束后
- 📴 先降低音量再拔耳机
- 🔌 长时间不用可拔下 USB
- 🧹 定期清洁耳机导管
- 📦 妥善收纳设备

## 🔧 常见问题快速解决

### 没有声音？
1. 检查 macOS 输出设备是否选择 "HiBy FC5"
2. 检查 FC5 屏幕是否显示音频格式
3. 检查耳机插头是否插紧（4.4mm 需用力按到底）
4. 尝试重新插拔 USB-C 线

### 声音太小？
1. 切换到 **Middle** 或 **High** 增益
2. 检查 macOS 系统音量
3. 检查网易云音乐音量
4. 确认使用 4.4mm 平衡口

### 有底噪或电流声？
1. 切换到 **Low** 增益
2. 检查 USB-C 线缆质量
3. 尝试更换 USB-C 接口
4. 检查是否有其他设备干扰

### 声音不够清晰？
1. 确认播放的是无损音乐
2. 尝试 FLTR: **Slow Lowpass**
3. 关闭 EQ（设为 OFF）
4. 清洁耳机导管和耳套

## 💡 Pro Tips

- 🎚️ **预热**: FC5 开机预热 10-15 分钟后音质更稳定
- 🔋 **电量**: USB 供电会消耗 MacBook 电量，建议插电使用
- 🎼 **音源**: 尽量使用 FLAC/APE 无损格式，而非 MP3
- 🎧 **平衡口**: 4.4mm 平衡口比 3.5mm 单端有更好的分离度
- 🔇 **安静环境**: 在安静环境下能听到更多细节
- 📊 **自定义 EQ**: 可用 HiByUSBMaster 工具精确调音（高级）
