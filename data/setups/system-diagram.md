## ⚡ 功率与驱动力分析

### 计算依据

**VULKAN2 参数**:
- 阻抗: 35Ω
- 灵敏度: 113dB/mW @ 1kHz

**HiBy FC5 输出**:
- 平衡输出电压: 4.0 Vrms
- 输出阻抗: ~2Ω

**理论最大输出功率** (4.4mm BAL):
```
P = V² / R = (4.0)² / 35 = 0.457W = 457mW
```

**可达声压级**:
```
SPL = 113dB + 10 × log₁₀(457) = 113 + 26.6 = 139.6 dB SPL (理论值)
```

### 实际使用分析

| 音量设置 | 估算功率 | 声压级 | 使用场景 |
|---------|---------|--------|---------|
| 20% | ~18mW | 126dB | 安静环境听音乐 |
| 40% | ~73mW | 132dB | 日常听音 ✅ 推荐 |
| 60% | ~165mW | 135dB | 大动态音乐 |
| 80% | ~293mW | 138dB | 接近最大音量 ⚠️ |
| 100% | ~457mW | 140dB | 危险音量 ❌ 不推荐 |

**结论**: 
- ✅ FC5 驱动力**充足**，可轻松驱动 VULKAN2
- ✅ 建议日常使用 **40-60%** 音量
- ⚠️ 超过 80% 可能损伤听力

## 🎚️ 阻抗匹配分析

### 阻尼系数 (Damping Factor)

```
DF = 负载阻抗 / 输出阻抗 = 35Ω / 2Ω = 17.5
```

**评价**: 
- ✅ **优秀匹配** (DF > 8 为良好)
- FC5 的低输出阻抗能很好地控制 VULKAN2 的动圈单元
- 保证良好的阻尼控制和低频响应

### 频响影响分析

输出阻抗对频响的影响:
```
△dB = 20 × log₁₀(1 + Z_out/Z_load)
     = 20 × log₁₀(1 + 2/35)
     = 20 × log₁₀(1.057)
     = 0.48 dB
```

**结论**: 频响偏差 <0.5dB，**几乎无影响** ✅

## 🔊 音质决定因素排序

```
1. 🎵 音源质量 (40%)
   └─ FLAC/APE > 高码率 MP3 > 普通 MP3

2. 🎧 换能器 (30%)
   └─ VULKAN2 八单元圈铁决定最终音色

3. 🎛️ DAC/耳放 (20%)
   └─ HiBy FC5 提供清晰、低失真的放大

4. 🔌 线材 (5%)
   └─ 4.4mm 平衡线，影响有限

5. ⚙️ 设置/调音 (5%)
   └─ GAIN/FLTR/EQ 微调
```

**优化建议优先级**:
1. 首先确保音源是无损格式
2. 正确设置 GAIN（避免底噪或驱动不足）
3. 选择合适的滤波器和 EQ
4. 最后考虑线材等玄学升级
