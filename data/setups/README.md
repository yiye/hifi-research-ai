# 播放系统配置说明

本目录存放个人音频播放系统的配置记录，包括音源设备、DAC/耳放、耳机/音箱的详细参数和推荐设置。

## 配置文件

### my-macbook-setup.json

**系统组成**:
- 🖥️ **音源**: MacBook Pro (Apple M4 Pro) + 网易云音乐
- 🎛️ **解码器**: HiBy FC5 屏幕版 USB DAC
- 🎧 **耳机**: 达音科 VULKAN 2（曜）八单元圈铁 IEM

**连接方式**:
```
MacBook Pro (USB-C) 
    ↓ [USB Audio]
HiBy FC5 (4.4mm BAL 输出)
    ↓ [4.4mm 平衡线]
VULKAN 2 (MMCX 接口)
```

## HiBy FC5 设置指南

### 🎚️ 增益设置 (GAIN)

| 选项 | 适用场景 | 说明 |
|------|---------|------|
| **Low** | 高灵敏度耳机（>110dB/mW）| ✅ **推荐用于 VULKAN2**，避免底噪 |
| **Middle** | 中等灵敏度耳机（95-110dB/mW）| 平衡模式 |
| **High** | 低灵敏度耳机（<95dB/mW）、高阻抗耳机 | 大推力模式 |

**VULKAN2 推荐**: Low（灵敏度 113dB/mW，非常容易驱动）

### 🔊 滤波器设置 (FLTR)

| 选项 | 特点 | 音质表现 |
|------|------|---------|
| **Fast Highpass** | 快速滚降高通 | 保留更多高频细节，瞬态快 |
| **Fast Lowpass** | 快速滚降低通 | 高频稍暗，适合亮声耳机 |
| **Slow Lowpass** | 慢速滚降低通 | ✅ **推荐**，自然声音，相位失真小 |
| **Slow Highpass** | 慢速滚降高通 | 平滑高频，适合长时间聆听 |

**VULKAN2 推荐**: Slow Lowpass（提供最自然、耐听的声音）

### 🎵 均衡器设置 (EQ)

| 预设 | 适用音乐类型 | 调音特点 |
|------|------------|---------|
| **OFF** | 所有类型 | ✅ **推荐**，保留 VULKAN2 原始调音 |
| **Vocal** | 流行、人声 | 突出人声，适合听歌 |
| **Bass** | 电子、嘻哈 | 增强低频量感 |
| **Treble** | 古典、器乐 | 提升高频清晰度 |
| **Pop** | 流行音乐 | 针对流行音乐优化 |
| **Classical** | 古典音乐 | 平衡动态和细节 |
| **Dynamic** | 大动态音乐 | 增强动态对比 |
| **Jazz** | 爵士乐 | 强调中频和氛围感 |
| **R&B** | R&B、灵魂乐 | 温暖低频 + 饱满人声 |
| **Metal** | 摇滚、金属 | 强劲低频 + 明亮高频 |

**VULKAN2 推荐**: 
- 日常听歌：**OFF** 或 **Vocal**（VULKAN2 本身已针对流行人声优化）
- 古典音乐：**Classical**
- 纯粹监听：**OFF**

### 📊 自定义 PEQ (高级)

如需精确调音，可使用：
- **Windows/Mac**: 下载 [HiByUSBMaster](https://www.hiby.com/support) 工具
- **Android**: 使用 HiByMusic App（设置中开启 USB 独占模式）

## 推荐配置方案

### 🎤 方案一：流行人声（日常听歌）
```json
{
  "gain": "Low",
  "filter": "Slow Lowpass",
  "eq": "Vocal" 或 "OFF",
  "volume": "40-60%",
  "output": "4.4mm BAL"
}
```
**适合**: 华语流行、日韩流行、欧美流行、人声为主的音乐

---

### 🎻 方案二：古典音乐
```json
{
  "gain": "Middle",
  "filter": "Slow Lowpass",
  "eq": "Classical",
  "volume": "45-65%",
  "output": "4.4mm BAL"
}
```
**适合**: 交响乐、室内乐、独奏音乐会

---

### 🎸 方案三：摇滚/金属
```json
{
  "gain": "Low",
  "filter": "Fast Lowpass",
  "eq": "Metal",
  "volume": "50-70%",
  "output": "4.4mm BAL"
}
```
**适合**: 摇滚、金属、朋克、硬核

---

### 🎹 方案四：纯净监听
```json
{
  "gain": "Low",
  "filter": "Slow Lowpass",
  "eq": "OFF",
  "volume": "按需调整",
  "output": "4.4mm BAL"
}
```
**适合**: 音乐制作、混音监听、发烧友纯净欣赏

## 音质优化建议

### macOS 系统设置

1. **打开 "音频 MIDI 设置" 应用**
   - 路径：`应用程序 > 实用工具 > 音频 MIDI 设置`
   - 选择 "HiBy FC5"
   - 设置格式：
     - 采样率：44.1kHz 或 48kHz（匹配音乐源）
     - 声道：2ch 立体声

2. **关闭系统音效**
   - `系统设置 > 声音 > 声音效果`
   - 取消勾选不必要的音效

3. **音量控制**
   - 建议使用 FC5 的硬件音量控制
   - macOS 系统音量设置为 100%
   - 通过 FC5 的音量键调节实际音量

### 网易云音乐设置

1. **音质选择**
   - 优先选择 **无损音质**（FLAC/APE）
   - 会员用户选择 **Hi-Res 音质**（如支持）

2. **输出设置**
   - 设置 > 播放
   - 输出设备：HiBy FC5
   - 关闭「音效增强」

3. **可选替代方案**
   - 考虑使用 **HiByMusic** 播放器（支持更高规格音频）
   - 或使用 **Audirvana**（专业音频播放器，支持比特完美输出）

### FC5 使用技巧

1. **预热**: 开机后预热 10-15 分钟，音质更稳定
2. **音量**: 从低音量开始，逐渐调高至舒适音量
3. **发热**: 正常使用会有轻微发热，属于正常现象
4. **电量**: USB 供电会消耗 MacBook 电量，建议插电使用
5. **接口选择**: 
   - 4.4mm 平衡口：更好的分离度、声场和动态
   - 3.5mm 单端口：便携场景或对比试听

## 听感参考

### VULKAN 2 声音特点

**整体风格**: 饱满温润、细腻耐听、人声靠前

**分频段表现**:
- **低频（5Hz-250Hz）**: 
  - 量感充足，下潜深
  - 双动圈（极低频 + 低频）提供强悍基础
  - 弹性好，速度适中
  
- **中频（250Hz-4kHz）**: 
  - 密度高，人声饱满靠前
  - 定制楼氏中频动铁保证质感
  - 人声位置靠前，适合流行人声
  
- **高频（4kHz-40kHz）**: 
  - 层次清晰，延伸自然
  - 四单元高频动铁提供丰富细节
  - 空气感充足，不刺激

**综合素质**:
- 解析力：⭐⭐⭐⭐☆
- 声场：⭐⭐⭐⭐☆（大声场）
- 分离度：⭐⭐⭐⭐☆
- 动态：⭐⭐⭐⭐☆
- 乐感：⭐⭐⭐⭐⭐

### 搭配评价：FC5 + VULKAN2

**匹配度**: ⭐⭐⭐⭐☆ (8.5/10)

**优点**:
- ✅ CS43198 DAC 音色中性偏暖，与 VULKAN2 搭配适中
- ✅ 平衡输出提供良好的分离度和声场
- ✅ 低输出阻抗（<2Ω）能很好控制 35Ω 阻抗
- ✅ 足够驱动力，VULKAN2 灵敏度高，FC5 驱动游刃有余
- ✅ 便携性好，适合外出使用

**可改进**:
- 🔸 如追求极致音质，可升级台式 DAC/耳放（如 Topping DX9）
- 🔸 网易云音乐音源质量参差不齐，建议使用无损音源

## 维护保养

### HiBy FC5
- 避免热插拔 USB 连接
- 保持 USB 和音频接口清洁干燥
- 长时间不用建议拔下 USB
- 正常发热无需担心，避免长时间高温环境使用

### VULKAN 2
- 定期清洁耳机导管和耳套（每周一次）
- 检查 MMCX 接口，避免频繁插拔导致松动
- 妥善收纳，避免线材打结或过度弯折
- 使用后放入保护盒，防潮防尘

## 安全提示

⚠️ **听力保护**:
- 首次使用前将音量调至最低
- 插拔耳机时不要佩戴，避免瞬间冲击声
- 长时间听音乐建议音量不超过 60-70%
- 每小时休息 10-15 分钟
- 避免在嘈杂环境中过度提高音量

⚠️ **设备保护**:
- 避免在极端温度环境使用
- 防止液体溅入设备
- 轻拿轻放，避免跌落

## 升级路径

如果未来想要提升音质，可以考虑：

1. **播放器升级**
   - Audirvana Studio (¥488)
   - Roon (订阅制)

2. **DAC/耳放升级**
   - 入门台式：Topping DX3 Pro+ (¥1000-1500)
   - 中端台式：Topping DX7 Pro+, iFi Zen DAC V2 (¥2000-3000)
   - 高端台式：Topping DX9, iFi Pro iDSD (¥5000-8000)

3. **线材升级**（主观因素较大）
   - 更换高级平衡线（¥300-2000）
   - 效果因人而异，建议试听后再决定

4. **耳机扩展**
   - 不同音色风格：如监听向的森海 IE600、暖声的 qdc Anole VX
   - 不同价位体验：预算充足可尝试旗舰 IEM
