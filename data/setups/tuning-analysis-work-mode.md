## 📝 摘要

通过对 VULKAN2 + FC5 组合进行精细的 PEQ 调校，最终形成的"工作模式"调音，在理念和频响特征上与水月雨 KADENZ（终章）高度一致。这个发现揭示了用户的听音偏好本质上是追求水月雨的 VDSF 调音哲学：**平衡、准确、自然、长时间耐听**。

## 🎧 背景：VULKAN2 的原始特征

### 设备信息
- **型号**: DUNU VULKAN 2（曜）
- **架构**: 8 单元圈铁混合（2DD + 6BA）
- **调音**: 饱满温润、人声靠前
- **特点**: 
  - 双动圈提供强劲低频
  - 四动铁高频延伸出色
  - 人声密度高，靠前

### 问题
由于 VULKAN2 的特性：
- ❌ 四动铁高频单元 + 人声靠前 → **齿音明显**（5-10kHz）
- ❌ 高频瞬态快速 → **容易刺激**（尤其对高频敏感型听者）
- ❌ 不适合长时间工作场景听音

## 🔧 调音历程：四个阶段的进化

### 版本 1.0 → 1.1: 滤波器优化（解决预振铃）
```
问题：线性相位滤波器的预振铃导致高频刺耳
解决：Slow Lowpass → Slow Highpass
效果：预振铃减少，但高频略暗
```

### 版本 1.1 → 1.2: 滤波器再优化
```
优化：Slow Highpass → Fast Highpass
理由：保留完整高频延伸 + 无预振铃
效果：✅ 清晰细腻不刺激
```

### 版本 1.2 → 1.4: EQ 和 PEQ 优化（齿音控制）
```
阶段 1: Vocal EQ → Pop EQ
- Vocal 增强 2-6kHz 导致齿音明显
- Pop 更平衡，齿音减少

阶段 2: 自定义 PEQ（精确控制）
- Band6: 6kHz, Peaking, Q=2.0, -2.5dB
- Band7: 8kHz, Peaking, Q=2.5, -2.0dB
效果：✅ 齿音完美控制
```

### 版本 1.4 → 1.5: 工作模式（高频整体柔化）⭐
```
新增：Band8: 4kHz, High Shelf, Q=0.7, -1.0dB
目的：整体高频柔化，提升长时间耐听度
用户反馈："听感好很多，也耐听多了"
效果：✅ 超强耐听度，适合 6-8 小时编程工作
```

## 📊 工作模式调音特征分析

### 频响曲线特征

```
频段              增益          特点
─────────────────────────────────────────────
20Hz - 1kHz      ±0.5dB       低频完全保留，清晰不闷
1kHz - 4kHz      0dB          中频原声，人声自然准确
4kHz - 6kHz      -1.0dB       高频开始温和衰减
6kHz             -3.5dB       齿音主频段精确控制
8kHz             -3.0dB       高频齿音控制
10kHz+           -1.0dB       超高频略微回升，保留空气感
```

### 调音理念

✅ **低频**: 保持原声，不增强不衰减  
✅ **中频**: 完全平坦，追求准确无染色  
✅ **高频**: 温和柔化，齿音精确控制  
✅ **整体**: 平滑、自然、耐听、不刺激  

### 使用场景

- 🎯 **主要用途**: 长时间编程工作听音（6-8+ 小时）
- 🎵 **音乐类型**: 纯音乐、Lo-Fi、轻音乐、古典、氛围音乐
- 💻 **工作特点**: 音乐不分散注意力，但能享受音质

## 💡 核心发现：调音理念的本质

### 用户通过 PEQ 调校追求的目标

```
1. 频响平衡
   ├─ 低频不轰头（平坦）
   ├─ 中频不染色（准确）
   └─ 高频不刺激（柔和）

2. 齿音控制
   ├─ 6-8kHz 精确衰减
   └─ 保留细节但不突兀

3. 长时间耐听
   ├─ 无刺激感
   ├─ 音乐不疲劳
   └─ 不分散注意力
```

### 这就是水月雨 VDSF 调音哲学！

**VDSF (Virtual Diffuse Sound Field) 调音理念**:
- 基于 HRTF（头部相关传递函数）
- 理想房间偏好优化
- 更均衡的能量分布
- 更真实的音色
- 自然不突兀的声音细节

**用户的 PEQ 调校 = 在追求 VDSF 理念 + 个体化调整**

## 🎯 调音公式总结

### VULKAN2 工作模式 ≈ KADENZ 原生调音

```
VULKAN2 (饱满人声向)
  ↓
+ Fast Highpass 滤波器（无预振铃）
  ↓
+ 齿音精确控制（6kHz -2.5dB, 8kHz -2.0dB）
  ↓
+ 高频整体柔化（4kHz High Shelf -1.0dB）
  ↓
≈ KADENZ 原生调音（VDSF + 个体差异补偿）
```

### 为什么需要 PEQ？

VULKAN2 和 KADENZ 的本质区别：

| 特征 | VULKAN2 | KADENZ |
|------|---------|--------|
| **架构** | 8 单元圈铁 (2DD+6BA) | 单动圈 |
| **调音倾向** | 人声靠前、饱满温润 | 中性监听、VDSF |
| **高频特性** | 四动铁，瞬态快、细节多 | 单动圈，自然柔和 |
| **齿音控制** | 原生齿音较明显 | 原生控制良好 |

**结论**:
- VULKAN2 通过 PEQ 调教后 ≈ KADENZ 原生
- PEQ 本质上是在"修正" VULKAN2 的圈铁特性
- 让其更接近单动圈的自然柔和

## 🎓 学到的经验

### 1. 调音理念比单一参数更重要

**错误认知**: "THD 低 0.0002% 就一定更好"  
**正确认知**: "调音理念和频响平衡比极限参数更影响听感"

### 2. 个体差异需要补偿

**发现**:
- 用户对高频敏感（属于 30-40% 敏感人群）
- 圈铁高频瞬态快，放大滤波器效应
- 需要个体化调整（High Shelf + 齿音控制）

### 3. 耐听度是长期使用的关键

**6-8 小时工作场景下**:
- 音质准确 < 耐听度
- 细节丰富 < 不刺激
- 高解析力 < 不疲劳

### 4. 单动圈的优势

**KADENZ 单动圈 vs VULKAN2 圈铁**:
- ✅ 相位一致性更好
- ✅ 音色更自然柔和
- ✅ 齿音控制更容易
- ✅ 长时间耐听度更佳
